How it works

xWTI is a Canopy appchain — a standalone blockchain that runs its own consensus and state machine, but inherits security from the Canopy root chain.

01

Canopy SDK

xWTI is built with the Canopy SDK. A Go plugin handles price polling and indicator computation.

02

RPC endpoints

The appchain exposes Tendermint RPC. Anyone can query the latest price, SMA, RSI, and block data — no API key required.

03

Real‑time feed

This site fetches live WTI prices from multiple sources. In production, you point it to your Canopy node's RPC.

⚡ Want to run your own oracle? Fork the repo and follow the tutorial.
